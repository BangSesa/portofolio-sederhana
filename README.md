# Website Sederhana HTML & CSS

Project ini merupakan website sederhana menggunakan **HTML dan CSS**
yang dibuat untuk latihan pembuatan tampilan website statis.

Website ini memiliki beberapa bagian seperti header, gallery,
services, dan sosial media.

---

# Fitur Website

- Landing page sederhana
- Gallery gambar
- Section layanan (services)
- Background image
- Icon sosial media
- Desain sederhana menggunakan CSS

---

# Teknologi yang Digunakan

- HTML5
- CSS3

---

# Struktur Project

website-sederhana-html-css

index.html → Halaman utama website  
style.css → File styling website  

assets:

background → Background website  
gallery1-6 → Gambar gallery  

icons:

fb → Facebook  
ig → Instagram  
yt → YouTube  

screenshoot → Screenshot tampilan website
